## Instructions

1. Create the `public/images/` directory if it doesn't exist
2. Add all the image files listed above
3. Ensure images are optimized for web (recommended: WebP or optimized JPG)
4. Recommended dimensions:
   - Service images: 600x400px or 600x800px
   - Class images: 600x400px
   - About image: 800x600px

The code is already configured to use these image paths. Once you add the images, they will automatically display.
